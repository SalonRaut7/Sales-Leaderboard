This is a **Sales Leaderboard** backend system built with **Node.js**, **Express**, and **MongoDB**. It aggregates sales data for multiple agents and generates a leaderboard ranked by **total sales amount**. The project also includes a simple frontend to visualize the leaderboard and add new sales data.

**Live Demo:** [https://sales-leaderboard.onrender.com](https://sales-leaderboard.onrender.com)
